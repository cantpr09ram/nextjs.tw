### 下載入門專案程式碼 (可選擇)

如果你沒有看過前面的課程，你可以透過下面的指令下載、安裝和執行本課的起始專案。這將創建一個有 `nextjs-blog` 目錄的專案，使其與上一課的成果相同。

如果您剛剛有完成前面的課程，則不需要進行這個步驟。

```bash
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/data-fetching-starter"
```
然後按照終端機輸出中的說明進行操作（`cd` 進入資料夾並啟動開發伺服器）。

你也應該要更新以下檔案：
- 將 `public/images/profile.jpg` 換成你的照片 （建議尺寸：400px X 400px）
- 將 `components/layout.js` 中的 `const name = '[Your Name]` 換成你的名子
- 將 `pages/index.js` 中的 `<p>[Your Self Introduction]</p>` 換成你的自我介紹
