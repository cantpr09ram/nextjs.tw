#### 下載程式碼（可選）

如果你不是從先前的課程學習至此，你可以下載、安裝並執行這一課程的程式碼，這會建立一個 `nextjs-blog` 目錄，使其與先前的課程結果相同。

再次強調，如果你剛完成先前的課程，這步驟是不必再做的。

```bash
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

接下來，按照指令輸出的指示操作（`cd` 進入目錄並啟動開發伺服器）。

你也應該更新下列檔案：

- 將 `public/images/profile.jpg` 改為你的照片（建議：400px 寬度/高度）。
- 將 `components/layout.js 內` 的 `const name = '[Your Name]'` 改為你的名字。
- 將 `pages/index.js 內` 的 `<p>[Your Self Introduction]</p>` 改為你的自我介紹。
