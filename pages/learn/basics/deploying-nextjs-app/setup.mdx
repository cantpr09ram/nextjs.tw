### 下載程式碼 (選用)

如果您不是接著上一個教學的話，您可以下載、安裝、並執行下方的程式碼。

這些程式碼設定了一個 `nextjs-blog` 資料夾，就跟上一個教學一樣。

再次提醒，如果你剛完成上一個教學的話，您可以跳過這個步驟。

```shell
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/basics-final"
```

接著依照輸出的指令操作（`cd` 進資料夾、啟動開發伺服器）。

你也應該要一起更新以下的檔案：

+ 將 `public/images/profile.jpg` 更新成你的照片 (建議: 400px 寬度/高度)。
+ 將 `components/layout.js` 檔案裡的 `const name = '[Your Name]'` 替換成你的名字。
+ 將 `pages/index.js` 檔案裡的 `<p>[Your Self Introduction]</p>` 替換成你的自我介紹。
